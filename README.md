facebook-tester
===============

##Requirements
```
ruby 2.1.3
opencv 
```

##Install
```
bundle install --path=vendor/bundle
```

##Usage
```
bundle exec ruby xxx.rb
```

##Memo
* Set ad with audience by AdsAPI (koala)
* Crawl own page to find our setting ad (mechanize)
* Use computer vision to detect our ad (opencv)
